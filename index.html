Sobre mim

Sou um desenvolvedor front-end apaixonado pelo que faço. Costumo dizer que “quem faz o que gosta vive de férias". É por isso que estou sempre querendo aprender algo novo que me ajude a ser uma pessoa melhor e melhor desenvolvedor. Tenho sólida experiência no domínio de tecnologias como HTML, CSS, JavaScript, ReactJS, Wordpress, entre outros. Estou sempre em busca de novos desafios e minha sede por aprender me impulsiona para criar experiências digitais funcionais e esteticamente agradáveis. Minha curiosidade e abordagem orientada a resultados me diferenciam na busca incessante por soluções criativas e inovadoras. Eu estou pronto para enfrentar qualquer projeto e contribuir para o sucesso da equipe.

Projetos

Formulário Multistep

Criação de formulário responsivo em React com o objetivo de avaliação da compra de um produto pelo comprador. Nesse projeto foram feitas a componetização de cada página, além do uso dos hooks, useState e useForm.

Tela de login

Criação de uma tela de cadastro e login em HTML, CSS e JavaScript. Neste projeto, a autenticação para o login foi implementada usando o LocalStorage, foram configurados eventos de clique para os botões de registro e login, e instruções condicionais "if" e "else" foram usadas para confirmar os caracteres inseridos e exibir mensagens de erro.

Jogo do Mario

Nesse projeto do jogo do Mario Bros foi utilizado as tecnologias HTML, CSS e Javascript. No Javascript foi utilizado arrow function, evento de click e if.

Conversor de Moedas

Criação de um conversor de moeda responsivo em HTML, CSS e JavaScript. Neste projeto, a troca de moeda e o cálculo dos valores de conversão foram realizados usando if e else, manipulação de DOM em JavaScript e um evento de clique no botão de conversão.

Kratos Portfolio

Kratos me pediu para criar esse portfólio para ele. Eu construí este portfólio usando HTML, CSS e JavaScript. Na parte JavaScript, empreguei funções e eventos de clique para abrir e fechar o menu mobile.

Trocador de cores com botão

O projeto "Trocador de cores com botão em javascript" é simples e interativo que permite aos usuários alterar a cor de um elemento específico na página, pressionando um botão.

Veterinarian Website

O projeto envolve a criação de um site moderno e funcional para uma clínica veterinária utilizando a última versão do framework Bootstrap - Bootstrap 5. O objetivo principal é desenvolver uma plataforma online que não apenas transmita profissionalismo e confiança, mas também forneça aos proprietários de animais de estimação com uma experiência agradável e informativa.
